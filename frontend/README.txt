COMO RODAR O FRONTEND:

1. NÃO USE npm start (este projeto não é React)
2. Use uma das opções abaixo:

OPÇÃO 1 (RECOMENDADA):
- Instale a extensão "Live Server" no VS Code
- Clique direito em index.html → "Open with Live Server"
- Acesse: http://localhost:5500

OPÇÃO 2:
- Abra o PowerShell na pasta frontend
- Execute: python -m http.server 5500
- Acesse: http://localhost:5500

OPÇÃO 3:
- Abra index.html direto no navegador (pode ter problemas de CORS)
